Git spurningar og svör

1. Hvað gera eftirfarandi Linux skipanir?
	cd        svar: "Change Directory" leyfir manni að færa sig úr Skrá yfir í aðra
	cd ~      svar: ~ er heima skráin
	ls        svar: "list files" sýnir skrár og skjöl
	pwd       svar: "Print Working Directory" sýnir í hvaða drifi maður er í
	mkdir     svar: "Make Directory" búa til önnur Skrá

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone     svar: Klónar gleymslu í skrá á tölvu
	git log       svar: til að sjá allar skráningar
	git status    svar: sér hvort skjal eða eitthvað hefur breyst áður en þú committar
	git diff      svar: sýnir breytingu á commits
	git checkout  svar: til að velja aðrar útgáfur

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff                   svar: sýnir breyting a skjalinu
	b) git diff --staged          svar: sýnur breytingu á staged hlutum
	c) git diff commit1 commit    svar: sýnir munin á tveim commits

4. 	Hvað er útgáfustýring (e.version control)?  svar: leyfir manni að skoða og breyta öðrum útgáfum af skjölum eða örðum hlutum, eins og git leyfir manni að gera

6.	Hversu oft telur þú að eigi að gera commit í verkefni, rökstuddu?
svar: eins oft og maður vill, nema það sé eitthvað sérstakt og má ekki eyðast eða maður vil eiga margar gerðir af einhvrju

7.	Hvað er "Working directory"?" "Staging area"?" og "Repository" í GIT?
svar: "Working Directory" er skráinn sem þú ert í, "Staging Area" er skrá sem inniheldur upplýsingar um næsta commit og "Repository" er staður sem geymir skjöl
