Git spurningar og svör

1. Hvað gera eftirfarandi Linux skipanir?
	cd        svar: "Change Directory" leyfir manni að færa sig úr Skrá yfir í aðra
	cd ~      svar: ~ er heima skráin
	ls        svar: "list files" sýnir skrár og skjöl
	pwd       svar: "Print Working Directory" sýnir í hvaða drifi maður er í
	mkdir     svar: "Make Directory" búa til önnur Skrá

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone     svar: Klónar gleymslu í skrá á tölvu
	git log       svar: til að sjá allar skráningar
	git status    svar: sér hvort skjal eða eitthvað hefur breyst áður en þú committar
	git diff      svar: sýnir breytingu á commits
	git checkout  svar: til að velja aðrar útgáfur
